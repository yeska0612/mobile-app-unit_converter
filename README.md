# 📱 Mobile App – Unit Converter

This project is an Android mobile application that converts different types of units.  
The app allows users to quickly convert values between different measurement systems.

---

## 🚀 Features

The application supports multiple unit conversions:

- Length (meter, kilometer, mile, inch)
- Mass (gram, kilogram, pound)
- Temperature (Celsius, Fahrenheit, Kelvin)
- Area
- Speed
- Volume
- Time
- Energy
- Density
- Angle

---

## 🛠 Technologies Used

- Kotlin
- Android Studio
- Gradle
- XML Layout
